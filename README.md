# vue-ssr-boilerplate

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and hot-reloads for development with SSR
```
npm run ssr:serve
```

### Compiles and minifies for production
```
npm run build
```

### Compiles and minifies for production
```
npm run ssr:build
```

### Starts for production
```
npm run start
```

### Starts for production with SSR
```
npm run ssr:start
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
